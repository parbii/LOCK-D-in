# **App Name**: AuthZen

## Core Features:

- Standard Sign-in: Sign-in page with email/password option.
- Password Reset: Forgot Password flow, to allow users to recover access.
- Google Sign-in/Sign-up: Sign-in/Sign-up with Google.
- Apple Sign-in/Sign-up: Sign-in/Sign-up with Apple.
- Facebook Sign-in/Sign-up: Sign-in/Sign-up with Facebook.
- Session Handling: Session management to persist logged-in state.
- AI Password Advisor: A tool to adaptively suggest security tips and strength ratings to improve users' password selections, based on login history or common vulnerability patterns.
- Dashboard: Users see a dashboard with community posts, goals, and habit streaks.
- Goal Creation and Tracking: Users can create goals and the progress bar increases exponentially as the daily habit streak grows.
- Mindset Modules: 10 mindset modules to shift mindset.
- Accountability Communities: A way to create private and public accountability communities.
- Chapter Leadership Application: Users can apply to lead a chapter on campus or at school after completing all 10 modules and submitting a 500 word essay.

## Style Guidelines:

- Primary color: Gray for neutrality and professionalism.
- Secondary color: Black for contrast and emphasis.
- Accent color: Hot pink for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, neutral, and easily readable interface.
- Use minimalist line icons for a modern and clean look. The icons should be related to security, privacy, and user accounts.
- Simple, centered layout for login/registration forms. Use clear, concise labels and hints.
- Subtle transition animations for form elements and feedback messages. Use a smooth, ease-in-out animation for all transitions.
