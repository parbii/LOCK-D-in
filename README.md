# LOCK-D-in

## Overview
Sync'd is a personal growth and productivity platform built to help students track their goals and engage with an accountability community. This MVP was developed as a passion project to bridge the gap between solo goal-setting and social support.

## Key Features
* **Goal Management:** Intuitive interface for defining and tracking daily/long-term milestones.
* **Community Hub:** A social feed designed for community engagement and shared progress updates.
* **Live Syncing:** Leverages Firebase for real-time data updates and secure user authentication.

## Tech Stack
* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Google Firebase (Authentication & Firestore)
* **Development:** VS Code, Git

## Why I Built This
As an MIS major, I wanted to create a solution that combines data-driven goal tracking and community engagement with a seamless user experience. This project allowed me to manage the full development lifecycle from initial "scaffold" to a functional MVP.
